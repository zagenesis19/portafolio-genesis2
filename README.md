# 🚀 Portafolio Profesional - Génesis Zapata

Este es mi portafolio web personal, diseñado para reflejar mi enfoque como **Arquitecta de Soluciones & Fullstack Developer**. Construido con una estética "Dark Tech" moderna, dinámica e inmersiva.

![Génesis Zapata Portfolio Mockup](https://raw.githubusercontent.com/withastro/astro/main/assets/press/astro-icon-light-gradient.svg) *(Logo Representativo)*

## 🛠️ Stack Tecnológico

- **Frontend:** [Astro](https://astro.build/) + [Tailwind CSS v4](https://tailwindcss.com/) + [Lucide Icons](https://lucide.dev/)
- **Backend & IA:** Python (FastAPI, OpenCV), PHP (Laravel)
- **Despliegue:** Vercel / Netlify
- **Animaciones:** Intersection Observer API & CSS Transitions

## ✨ Características Principales

- **Arquitectura Escalable:** Estructura de componentes reutilizables en Astro.
- **Dark Tech Design:** Interfaz inmersiva con efectos de "Glassmorphism" y gradientes dinámicos.
- **Scroll Reveal:** Animaciones fluidas que se activan al navegar por las secciones.
- **Ecosistema Visual:** Stack tecnológico categorizado con logos interactivos.
- **Responsive Design:** Optimizado para dispositivos móviles con un menú interactivo.

## 🚀 Instalación Local

Si deseas clonar y probar el proyecto localmente:

1. **Clonar repositorio:**
   ```bash
   git clone https://github.com/zagenesis19/portafolio-genesis.git
   ```
2. **Instalar dependencias:**
   ```bash
   npm install
   ```
3. **Ejecutar servidor de desarrollo:**
   ```bash
   npm run dev
   ```
